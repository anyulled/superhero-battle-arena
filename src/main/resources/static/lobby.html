Register Team via API

Team registration is currently handled exclusively via the API.

# Example cURL Request

POST /api/teams/register

curl -X POST \

"http://localhost:8080/api/teams/register?name=YourTeamName&members=Member1,Member2"

Endpoint: /api/teams/register

Method: POST

Parameters:

  • name: Your team name (String)
  • members: Comma-separated member names (List<String>)

Explore the full API documentation at Swagger UI Documentation
